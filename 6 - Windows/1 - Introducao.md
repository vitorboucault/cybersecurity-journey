# Windows

Cada usuário no windows tem um determinado nível de acesso a recursos dentro da maquina. 

O Windows separa os usuários em alguns tipos:

- Guest (Convidado)
  - Conta restrita com permições minimas
- Standard
  - Conta para as tarefas do dia a dia
- Administrator:
  - Conta com privilégios elevados, pode mudar configurações, intalar softwares eles tem controle total do sistema

## Windows Desktop (Área de Trabalho)

Ao logar temos acesso a duas áreas diferentes:

- Área de trabalho Desktop
- Taskbar

### Start Menu

Menu de acesso rápido onde podemos encontrar pastas, programas e arquivos de forma rápida.

## Built-in Tools 

São as ferramentas nativas do Windows, como o Note Pad, explorador de arquivos etc

## Informações do Sistema

Podemos ter informações do sistema em **Settings** e na seção **About Your PC**

## Organização e explroação de arquivos

O windows organiza o sistema de arquivos por meios de pastas de forma hierarquica.
