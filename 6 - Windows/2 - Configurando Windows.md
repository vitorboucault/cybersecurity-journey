# Configurando e protegendo Windows

É necessário aprender a atualizar, desinstalar e configurar aplicativos no Windows

## Atualizando aplicações

- Windows Update
  - Serve para atualizar o Windows e conferir se tem alguma atualização importante
  - Pode ser acessado pelo app Settings
- Atualizando aplicações
  - Cada software tem sua forma de ser atualizado
  - Aplicações nativas são atualizadas em segundo plano

## Instalando aplicações

Pode ser pela Microsoft Store ou pela internet

## Desinstalando aplicações

- Pelo app settings do windows é possível desinstalar no Painel De Controle
- Pode ser pelo Microsoft Store também

## Configurações

O Windows settings possui muitas configurações que podem ser feitas e o painel de controle também

## Gerenciador de Tarefas

Aplicação nativa para ver processos, consumo de CPU, memoria ram, detalhes e serviços
