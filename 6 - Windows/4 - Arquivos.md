# Arquivos

No sistema de arquivos NFTS do Windows, os arquivos e pastas possuem tipos de permissões diferentes como:


- Ler (Read)
  - Pastas: Permite ver ou listar arquivos ou subpastas
  - Arquivo: Permite ver ou acessar conteúdos dos arquivos
- Escrever (Write)
  - Pastas: Permite adicionar arquivos e subpastas
  - Arquivos: Permite escrever em um arquivo
- Controle total
- Modificar
- Ler & Executar
- Listar conteudo das pastas
