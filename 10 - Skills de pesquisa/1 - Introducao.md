# Introdução

Aqui vamos aprender ferramentas de pesquisa para conseguir encontrar a informação da maneira correta.

Vamos aprender a:

- Avaliar fontes de informação
- Usar as engines de pesquisa de forma eficiente
- Explorar as engines de pesquisa especializadas
- Ler documentação tecnica
- Fazer uso de redes sociais
- Consultar meios de comunicação
