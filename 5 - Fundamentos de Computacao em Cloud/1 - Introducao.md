# Fundamentos de Cloud

Computação em cloud permite que você rode aplicações de forma eficiiente em uma infraestrutura compartilhada que te permite fazer alterações de forma rápida e fácil quando necessário.

## Beneficios

A computação em cloud veio pra resolver problemas de custo, capacidade limitada, crescimento lento.

Aqui são alguns beneficios

- Escalabilidade
- Serviço on-demand: usa quando precisar e por conta propria
- Paga o que usar
- Segurança
- Alta disponibilidade: a qualquer momento voce pode acessar os seus dados
- Acesso global: qualquer pessoa pode acessar a aplicação de qualquer pais
