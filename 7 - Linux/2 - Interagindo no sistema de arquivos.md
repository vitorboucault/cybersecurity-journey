# Filesystem

## Comandos básicos

### Onde estou?

Mostra pasta de onde o terminal está no momento: `pwd`

### Comando ls

- Mostra as pastas do diretório: `ls`

- Mostra as pastas de forma organizada: `ls -l`

- Lista os arquivos todos os e organiza eles: `ls -al`

### Comandos para navegar

- Vai direto para a pasta `cd <diretorio>`

- Volta uma pasta `cd ..`

### Comando "encontrar"
- Localiza arquivos: `find`
- Localiza arquivos a partir do nome: `find <pasta inicial> -name <nome do arquivo>`
  - Pode incluir o ~ como o diretorio home
              - Ex.: `find ~ -name mission_brief.txt`

### Comando para exibir arquivos

- `cat arquivo.extensão`

### Comando pwd

- Exibe o diretório onde está trabalhando: `pwd`

### Cria arquivos e pastas

- Comando para criar arquivos: `touch`
- Comando para criar pastas: `mkdir`

### Removendo arquivos e pastas

- Comando para remover arquivos: `rm arquivo`
- Comando para remover diretórios: `rm -R diretorio`

### Copiando e movendo arquivos

- Copia todo o conteudo de um arquivo para outro: `cp arquivoOriginal arquivoCopiado`
- Move ou renomeia arquivos para pastas: `mv arquivo <pasta ou novo nome>`

### Determinando tipo de arquivo

- Revela o tipo de arquivo: `file <arquivo>`
