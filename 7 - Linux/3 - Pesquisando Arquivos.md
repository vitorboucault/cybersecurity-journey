# Pesquisando arquivos

## Comando Find

- `find -name nome-do-arquivo`: comando serve para encontrar direto arquivos necessários sem precisar ficar usando cd e ls.
- Caso não saiba o nome do arquivo, pode usar da seguinte forma: `find -name *.txt`

## Comando Grep

Permite pesquisar conteúdos de um arquivo ou valores especificos

- Utilizando `grep "conteudo mensagem" arquivo.txt`
- Ou utilizando `grep -R "palavra especifica" /diretorio/`
