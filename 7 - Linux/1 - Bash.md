# Bash

## Comandos básicos

- Onde estou?
  - Mostra pasta de onde o terminal está no momento
    - pwd
- Mostrar pastas do diretorio
  - ls
  - ls -l 
    - Lista os conteúdos
  - ls -al
    - Lista arquivos escondidos
- Comandos para navegar
  - cd <<**diretorio**>>
    - vai direto para a pasta
  - cd ..
    - Volta uma pasta
- Comando "encontrar"
  - find
    - Localiza arquivos
  - find <**pasta inicial**> -name <**nome do arquivo**>
    - Pode incluir o ~ como o diretorio home
      - Ex.: find ~ -name mission_brief.txt
- Comando para exibir arquivos
  - cat -arquivo-

## Comandos pra investigar o sistema

- Quem sou eu?
  - whoami
    - Mostra o username
- Qual o sistema está usando?
  - uname
  - uname -a
- Confere armazenamento em disco
  - df -h
    - -h significa que pode ser lido por humano
