# Diretórios importantes

1. `/etc` -> configurações e credenciais
2. `/var/log` -> logs do sistema
3. `/root` --> é a home para o usuario root.
3. `/tmp` -> usado por malware para armazenar arquivos temporários, qualquer usuário pode escrever lá

| Pasta  | O que guarda                                         |
|--------|------------------------------------------------------|
| /      | Raiz — tudo começa aqui                              |
| /bin   | Comandos essenciais (ls, cp, mv...)                  |
| /boot  | Arquivos pra iniciar o sistema                       |
| /dev   | "Arquivos" que representam dispositivos (HD, USB...) |
| /etc   | Configurações do sistema e aplicativos               |
| /home  | Pasta pessoal de cada usuário (/home/joao)           |
| /lib   | Bibliotecas essenciais do sistema                    |
| /media | Onde mídias externas (USB, CD) são montadas          |
| /opt   | Softwares de terceiros instalados manualmente        |
| /root  | Pasta pessoal do usuário root (administrador)        |
| /sbin  | Comandos administrativos do sistema                  |
| /tmp       | Arquivos temporários (apagados no boot!)             |
| /usr| Programas e bibliotecas dos usuários                                                     |
|/var | Dados variáveis: logs, e-mails, arquivos de apps web                                                     |
