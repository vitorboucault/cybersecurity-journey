# Introdução

Um banco de dados é um lugar onde podemos armazenar informações, é como se fosse um caderno com páginas que não acabam.

As informações em um banco de dados são armazenadas em tabelas.

## Tabelas, Colunas e Linhas

Uma tabela é organizada com linhas e colunas.

Colunas são os titulos no topo da tabela, e as linhas contém um set completo de informações

## Queries em um DB

São perguntas especificas para o banco, por exemplo:

1. Quanto um produto já vendeu?
2. Pra quantas pesssoas?
3. Me mostra o produto mais barato
