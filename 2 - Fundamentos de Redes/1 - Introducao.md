# Introdução a redes

Uma rede é uma coleção de dispositivos interconectados que podem se comunicar enviando dados e compartilhando recursos entre si.

**Principais conceitos:**

1. Nodes: dispositivos individuais conectados a uma rede
2. Links: caminhos de comunicação que conectam nomes via cabo ou wireless
3. Data Sharing: o proposito principal de uma network permitir a troca de dados

**Por que as networks são importantes?**

1. Compartilhamento de recursos
2. Comunicação
3. Acesso de dados
4. Colaboração

**Tipos de redes:**

**1. Local Area Network (LAN):**
    1. Dispositivos conectados em uma curta distancia
    2. De propriedade de uma pessoa ou empresa
    3. Alta velocidade de transferencia de dados
    4. Conectados por cabo ethernet ou Wi-Fi.
**2. Wide Area Network (WAN):**
    1. Se expande por cidades, países e continentes
    2. Conecta as LANs
    3. Propriedade coletiva
    4. Transferencia de dados é devagar comparados com as LANs
    5. Utiliza fibra ótica, satelites e linhas de comunicação alugadas

**Como as LANs e WANs trabalham juntas?**

LANs podem se conectar a WANs. Quando se acessa a internet, uma Home LAN se conecta a um provedor de internet.

O modem atua como uma ponte entre a a rede de casa e o provedor de internet
