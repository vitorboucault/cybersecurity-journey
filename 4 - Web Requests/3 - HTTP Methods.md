# HTTP Methods

Os métodos são usados para falar ao servidor o tipo de request que vai ser feito e como responder.

**Request Methods**

- GET —> Faz o request de um recurso especifico, ou seja, consegue uma informação do servidor
- POST —> Envia dados para o servidor para criar algo no servidor
- HEAD —> Solicita o HEADER que vai ser retornado
- PUT —> Atualiza um recurso no servidor, permitir esse metodo sem os controles apropriados pode dar espaço para uploads maliciosos.
- DELETE —> Deleta um recurso existente no webserver.
- OPTIONS —> Retorna informação sobre o servidor
- PATCH —> Aplica modificações parciais do recurso a uma localização especifica
