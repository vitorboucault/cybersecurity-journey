# Web Server

Um servidor web é um software que utiliza um protocolo HTTP para entrega requisições feitas a ele

## Virtual Hosts

Servidores Web podem ser hosts de multiplos sites, com diferentes dominios. Para conseguir isso eles usam virtual hosts.

É basicamente uma configuração em texto. Ao ter uma requisição para um web server ele vai direcionar pro website correto que foi solicitado.

Não existe um numero limite para os virtual hosts
